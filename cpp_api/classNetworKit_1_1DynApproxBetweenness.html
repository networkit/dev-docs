class NetworKit::DynApproxBetweenness : public NetworKit::Centrality, public NetworKit::DynAlgorithm

Interface for dynamic approximated betweenness centrality algorithm.

Public Functions

DynApproxBetweenness(const Graph &G, double epsilon = 0.01, double delta = 0.1, bool storePredecessors = true, double universalConstant = 1.0)

The algorithm approximates the betweenness of all vertices so that the scores are within an additive error epsilon with probability at least (1- delta). The values are normalized by default.

Parameters

  • G – the graph

  • storePredecessors – keep track of the lists of predecessors?

  • epsilon – maximum additive error

  • delta – probability that the values are within the error guarantee

  • universalConstant – the universal constant to be used in computing the sample size. It is 1 by default. Some references suggest using 0.5, but there is no guarantee in this case.

virtual void run() override

virtual void update(GraphEvent e) override
