class NetworKit::PageRank : public NetworKit::Centrality

Compute PageRank as node centrality measure. NOTE: There is an inconsistency in the definition in Newman’s book (Ch. 7) regarding directed graphs; we follow the verbal description, which requires to sum over the incoming edges (as opposed to outgoing ones).
