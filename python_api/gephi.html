networkit.gephi.csv module

networkit.gephi.csv.exportNodeValues(values, fpath, column_name)

This method exports node values (e.g. community information, betwenness centrality values) to a CSV file. The values can then be imported to Gephi. :param - values: python list or Partition object that contains the values to be exported. :param - fpath: the file including its path to be written to :param - column_name: name of the column as description for the data
